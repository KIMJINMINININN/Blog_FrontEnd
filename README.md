## New Blog FrontEnd
### Tech
```
1. TypeScript
2. React.js, Next.js
3. Redux
4. React-Query
5. Styled-Component
6. Jest
7. Git, GitHub
8. Visual Studio Code
```

### Setting

1. Clone this repository
2. yarn package update

```
    yarn
```

### How to use

to start dev server

```
    yarn dev
```

build

```
    yarn build
```

start

```
    yarn start
```
